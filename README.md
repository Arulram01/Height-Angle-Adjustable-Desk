
# Ergonomic Height & Angle Adjustable Desk 

## 📖 Project Overview
This repository contains the design, 3D models, 2D technical drawings, and documentation for an innovative **Height and Angle Adjustable Desk**. The desk is designed to provide ergonomic benefits by allowing users to seamlessly modify both the height and the tilt of the work surface. 

By accommodating various working positions (including sitting and standing), this workstation aims to improve user posture, reduce physical strain, and enhance overall productivity.

## ✨ Key Features (Concept F Final Design)
After a rigorous conceptualization and screening process using Pugh's Concept Selection, **Concept F** was selected as the optimal design. Key features include:
* **Smooth Height Control:** Utilizes a bevel gear-based height adjustment system operated by a hand lever for precise, low-effort transitions.
* **Secure Angle Adjustment:** Features a toothed bar mechanism with a reliable locking system to prevent accidental movement and maintain stability.
* **Integrated Lighting:** Includes a built-in flexible table lamp for focused visibility.
* **Mobility & Stability:** Built on a sturdy base equipped with caster wheels for easy relocation.
* **Laptop Platform:** Features a central rotating disk/platform for flexible laptop positioning.

## 📊 Technical Specifications
The design was optimized to meet strict customer needs and engineering metrics:
* **Height Adjustment Range:** 60 cm to 120 cm 
* **Angle Adjustability:** 0 to 60 degrees 
* **Load Capacity:** ≥ 50 kg 
* **Adjustment Time (One-handed):** ≤ 5 seconds 
* **Target Manufacturing Price:** 45,000 LKR 

## 🗂️ Repository Contents
* `/3D_Models`: Contains the final 3D CAD assembly files for the desk.
* `/2D_Drawings`: Includes detailed 2D component drawings (Plans, Elevations, Leg Parts, Gears, etc.).
* `/Documentation`: Contains the comprehensive Final Project Report detailing the market opportunity, Affinity Diagrams, QFD method, and Concept Generation .

## 🛠️ Development Process highlights
1. **Opportunity Identification:** Targeted the high-demand market of home offices and educational sectors needing ergonomic solutions.
2. **Customer Need Analysis:** Utilized surveys to build an Affinity Diagram and Analytical Hierarchy Process (AHP) to prioritize usability, cost, and safety.
3. **QFD Matrix:** Translated customer needs (like "Easy Adjustments" and "High Load Capacity") into measurable technical specifications.
4. **Concept Generation & Selection:** Drafted six distinct concepts (A through F), ultimately refining Concept F for its superior handling, safety, and durability scores
