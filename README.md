# Ergonomic Height & Angle Adjustable Desk 

## 📖 Project Overview
[cite_start]This repository contains the design, 3D models, 2D technical drawings, and documentation for an innovative **Height and Angle Adjustable Desk**[cite: 8, 420]. [cite_start]The desk is designed to provide ergonomic benefits by allowing users to seamlessly modify both the height and the tilt of the work surface[cite: 8]. 

[cite_start]By accommodating various working positions (including sitting and standing), this workstation aims to improve user posture, reduce physical strain, and enhance overall productivity[cite: 9].

## ✨ Key Features (Concept F Final Design)
[cite_start]After a rigorous conceptualization and screening process using Pugh's Concept Selection, **Concept F** was selected as the optimal design[cite: 351, 353, 358]. Key features include:
* [cite_start]**Smooth Height Control:** Utilizes a bevel gear-based height adjustment system operated by a hand lever for precise, low-effort transitions[cite: 345, 359].
* [cite_start]**Secure Angle Adjustment:** Features a toothed bar mechanism with a reliable locking system to prevent accidental movement and maintain stability[cite: 345, 360].
* [cite_start]**Integrated Lighting:** Includes a built-in flexible table lamp for focused visibility[cite: 339, 346, 388].
* [cite_start]**Mobility & Stability:** Built on a sturdy base equipped with caster wheels for easy relocation[cite: 346].
* [cite_start]**Laptop Platform:** Features a central rotating disk/platform for flexible laptop positioning[cite: 324, 328].

## 📊 Technical Specifications
[cite_start]The design was optimized to meet strict customer needs and engineering metrics[cite: 142, 197]:
* [cite_start]**Height Adjustment Range:** 60 cm to 120 cm [cite: 198]
* [cite_start]**Angle Adjustability:** 0 to 60 degrees [cite: 198]
* [cite_start]**Load Capacity:** ≥ 50 kg [cite: 198]
* [cite_start]**Adjustment Time (One-handed):** ≤ 5 seconds [cite: 198]
* [cite_start]**Target Manufacturing Price:** 45,000 LKR [cite: 206]

## 🛠️ Development Process highlights
1. [cite_start]**Opportunity Identification:** Targeted the high-demand market of home offices and educational sectors needing ergonomic solutions[cite: 99, 136].
2. [cite_start]**Customer Need Analysis:** Utilized surveys to build an Affinity Diagram and Analytical Hierarchy Process (AHP) to prioritize usability, cost, and safety[cite: 146, 175].
3. [cite_start]**QFD Matrix:** Translated customer needs (like "Easy Adjustments" and "High Load Capacity") into measurable technical specifications[cite: 191].
4. [cite_start]**Concept Generation & Selection:** Drafted six distinct concepts (A through F), ultimately refining Concept F for its superior handling, safety, and durability scores[cite: 316, 344, 354].
